# Clevo W650KL 黑苹果相关文件

## 简介
这是蓝天 Clevo W650KL 准系统（国内具体型号为炫龙 Shinelon 毁灭者 KL-4681HN3荣耀版)的黑苹果相关文件, 从 High Sierra 10.13.4 开始适配。

- 硬盘 : HGST HTS541010B7E610 5400转 1TB Sata 磁盘
